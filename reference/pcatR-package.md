# pcatR: Analyze Pragmatic Context Assessment Tool Data

Tools for importing, validating, classifying, summarizing, comparing,
and visualizing responses to the 14-item Pragmatic Context Assessment
Tool (pCAT). The package preserves the separate direction and effect
responses and does not calculate or claim a validated total scale score.

## Details

The package supports the original CFIR mapping, the updated 2022 CFIR
mapping, team agreement diagnostics, repeated assessments,
barrier-focused action-plan tables, and an optional Shiny analysis
dashboard.
