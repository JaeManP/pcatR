# Run the built-in pcatR self-test

Runs checks of the instrument dictionary, validation, classification,
summaries, reshaping, longitudinal comparison, action planning, the
standard analysis wrapper, and plotting.

## Usage

``` r
pcat_self_test(verbose = TRUE)
```

## Arguments

- verbose:

  Print one line per check.

## Value

A data frame with check names and results, invisibly.
