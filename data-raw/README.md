# Source data notes

`inst/extdata/pcat_items.csv` transcribes the 14-item instrument and records
original and updated CFIR mappings. Exact item wording is attributed under CC
BY 4.0; see `LICENSE.note`.

`inst/extdata/pcat_eric_candidates.csv` transcribes checkmarks and Tier 1
shading from Table 3 of Robinson and Damschroder (2023). The source table uses
`Relative Priority`, whereas pCAT item 12 uses `Relative Advantage`; this is
marked as an approximate, non-identical link and is excluded by default.

`inst/extdata/pcat_example.csv` is entirely synthetic.
