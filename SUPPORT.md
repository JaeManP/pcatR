# Support

Use GitHub Discussions for usage questions and GitHub Issues for
reproducible software defects. Before filing an issue, run:

``` r

packageVersion("pcatR")
pcat_self_test()
sessionInfo()
```

Include a minimal de-identified example created with
[`data.frame()`](https://rdrr.io/r/base/data.frame.html) or synthetic
data. Never attach protected health information, direct identifiers,
access tokens, or confidential free-text comments.

Methodological requests that would introduce a total pCAT score, change
item wording, or alter source mappings require a published rationale and
explicit versioned review.
